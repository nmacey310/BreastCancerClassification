Methodology

  1. Explan your ML and DL metodology

This project uses both machine learning and deep learning techniques to classify breast tumors as either malignant or benign using the Breast Cancer Wisconsin Diagnostic dataset.

  1. Introduce the topics you used in your project

Exploratory Data Analysis (EDA): Understand the dataset, check for missing values, and explore feature distributions.Summary statistics were generated to examine central tendencies and variability of the 30 numerical features. A correlation heatmap highlighted relationships and redundancies between features. Visualizations, including boxplots and class distribution charts, confirmed that the dataset is reasonably balanced, with 357 benign and 212 malignant samples, making it suitable for model training. EDA provided insight into which features might be most informative for tumor classification

Machine Learning Models: Implement models like Random Forest and SVM for tumor classification. Random Forest - Captures nonlinear patterns and identifies feature importance. RF achieved a test accuracy of 95.6%, and the top features contributing to classification were visualized. Support Vector Machine (SVM)-An RBF-kernel SVM was used due to its ability to model complex decision boundaries. SVM achieved the highest test accuracy of 98.2%, demonstrating excellent generalization.

Hyperparameter tuning was performed using GridSearchCV and validation curves, optimizing parameters such as n_estimators for RF, which improved performance and ensured robust model selection.

Deep Learning Models: Build a feed-forward neural network A simple feed-forward neural network was created using Keras with two hidden layers (32 and 16 neurons). EarlyStopping was added to prevent overfitting by monitoring validation loss.The neural network achieved a test accuracy of 95.6%, comparable to Random Forest, showing that DL can provide reliable predictions even on smaller structured datasets.

Visualization & Interpretation: Confusion matrices, ROC curves, feature importance. Models were evaluated using accuracy, precision, recall, F1-score, confusion matrices, and ROC-AUC curves. Feature importance plots from RF provided interpretability, while ROC curves for RF and SVM confirmed strong discriminative ability. Together, these evaluation methods gave a comprehensive understanding of model performance and reliability

learning_rates = [0.001, 0.01]
best_val_acc = 0
best_lr = None

for lr in learning_rates:
    temp_model = Sequential([
        Dense(32, activation='relu', input_shape=(X_train.shape[1],)),
        Dropout(0.2),
        Dense(1, activation='sigmoid')
    ])
    temp_model.compile(optimizer=Adam(learning_rate=lr), loss='binary_crossentropy', metrics=['accuracy'])
    
    early_stop = EarlyStopping(monitor='val_loss', patience=10, restore_best_weights=True)
    
    history = temp_model.fit(X_train, y_train, validation_split=0.2, epochs=50, batch_size=32,
                             callbacks=[early_stop], verbose=0)
    
    val_acc = max(history.history['val_accuracy'])
    print(f"Learning rate: {lr}, Best Val Accuracy: {val_acc:.4f}")
    
    if val_acc > best_val_acc:
        best_val_acc = val_acc
        best_lr = lr

print("Best NN learning rate:", best_lr)

Conclusions

This project demonstrates that machine learning models can effectively classify breast cancer tumors with high accuracy using the Breast Cancer Wisconsin Diagnostic dataset. Support Vector Machine (SVM) achieved the best performance with approximately 98% accuracy, showing its strength on structured, high-dimensional data. Random Forest also performed strongly at around 96% accuracy and provided valuable interpretability through feature importance analysis. The feed-forward neural network achieved comparable accuracy 95.6%, but slightly underperformed compared to SVM. This outcome reflects the limitation of deep learning models on smaller structured datasets. Visualizations such as ROC curves, confusion matrices, and correlation maps gave deeper insight into model behavior and dataset structure.

Overall, the results show that traditional machine learning algorithms remain highly effective for medical diagnostic datasets and can provide excellent performance with minimal tuning.
